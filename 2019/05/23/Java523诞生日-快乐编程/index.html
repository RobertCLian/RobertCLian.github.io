Posted on 2019-05-23 | In web后端

1995年5月23, 由当时任职SUN微系统的James Gosling等人撰写的跨平台面向对象的设计语言Oak以Java的名称正式发布了。

注重语法的注释

来自绝对菜鸟的注释

// I am not sure if we need this, but too scared to delete.

中文：个人不确认是不是需要，但是实在不敢删除

来自正直程序员的注释

中文：个人不负责这块的质量，因为他们逼迫我违心的写了这段代码

来自电影的注释

中文：疯了吧？这是斯巴达！

绝对尽责的注释

中文：给变量i增加一个记数

绝对会被忽略的注释

中文：谁在意？

绝对不能信任注释

中文：返回为true （编辑：永远不能相信注释）
